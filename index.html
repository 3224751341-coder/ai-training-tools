AI 工具使用指南

不是"学一个 AI 就够了"——每个工具各有所长，搭配使用才是正解。

核心逻辑

现在的 AI 工具很多，不同工具特性和强项不一样。不存在"最好的 AI"——只有和你的工作内容最匹配的 AI。

关键思路：不同 AI 互为杠杆。A 工具产出给 B 工具用，B 工具的结果拿到 C 工具去打磨。一条链串起来，每个人都能搭出自己的 AI 工作流。

更重要的是：AI 发展很快，但垂直领域的落地还很少。最宝贵的资源是日常工作里的真实问题——80% 现在都能用 AI 解决。关键在于先有问题意识：没有具体问题，AI 就落不了地。问题会变，AI 也会变，所以最重要的不是学工具，而是持续去想——我手头哪些事，可以让 AI 来干？

四件工具
01

ChatGPT

海报设计首选，生图能力顶级

擅长

  • GPT-4o 原生生图，文字渲染和排版精准
  • 简单 prompt 就能出好图——GPT 聪明，会自己补细节
  • 想更好就多轮沟通、持续打磨
  • 记忆功能：遇到好风格说"记住这个"，之后保持一致性
  • 可记品牌色、Logo 位置、受众、常用尺寸
  • 日常文字聊天无限额

注意

  • 生图有每日限额
  • 想出精品需要多轮打磨，不是一句出片
习惯之后：你越用，GPT 越懂你。限额不够 → 注册新号，让老 GPT 输出偏好摘要，喂给新号即可无缝迁移。
💻 桌面端下载

Mac: App Store 下载  |  Windows: 官方下载页

02

Gemini

免费万能入口，截图即问

擅长

  • 完全免费，问问题、截图、对话全无限制
  • 响应极快
  • 上下文超大（1M token），整份长文档直接扔进去分析
  • 遇到不知道怎么解决的问题 → 截图丢给她问怎么办

注意

  • 生图差，不适合做视觉产出
杠杆定位：Gemini 是"解题入口"——卡住了先问她怎么办，拿到思路，再用别的 AI 去执行。
💻 桌面端使用

Web: gemini.google.com  |  移动端: iOS / Android  |  桌面端: Chrome 浏览器打开后点地址栏右侧「安装」图标即可变成桌面 App

03

Agent 类工具 （Qoder · WorkBuddy）

一句话办事，AI 替你操作电脑

擅长

  • 自然语言驱动——"帮我把这份数据做成表格"直接出结果
  • 批量处理文件、生成文档/PPT/表格、数据分析
  • WorkBuddy 支持微信/企微远程操控
  • WorkBuddy 国内 AI 办公 Agent 月活第一，定位非技术用户

注意

  • 学习成本偏高，初期需要摸索
  • 目前阶段建议：团队里 1–2 个人先跑通，跑出效果再推广
杠杆定位：Agent 是"执行层"——把 ChatGPT/Gemini 的智力 + 实际操作能力合在一起。你说"做什么"，它自己去规划步骤、调不同能力、操作软件。
04

DeepSeek + Claude Code 付费

复杂项目一站式交付，自由度和上限最高

擅长

  • 直接操作电脑——读文件、写文件、跑命令，全自动
  • DeepSeek 模型便宜，token 成本极低
  • 一句话需求 → 自动规划 → 执行 → 检查，闭环交付
  • 产出形式不限：PDF、网页、系统、自动化脚本
  • 已有成品：查房字典 2.0、房态系统、视频处理 pipeline、各类操作手册

注意

  • 需要花钱（API + Claude Code 订阅），但成本相对功能来说很值
  • 终端/命令行操作，普通同事门槛较高
  • 初始配置需要搭环境
杠杆定位：工具链的"最后一环"——ChatGPT 出设计、Gemini 解难题、Agent 跑流程，DeepSeek + Claude Code 整合成最终产品（系统、报表、自动化）。
实例：Iglu Sydney 房态系统

8 栋学生公寓的实时房态自动抓取 → 网页集中展示 → 定时自动更新。从数据抓取、网页布局、自动刷新到部署，全程由 DeepSeek + Claude Code 完成。以前需要人工每天手动查看 8 次的工作，现在变成自动运行、随时可查的网页面板。

串联起来

你的 AI 工具链

🤔 卡住了 → Gemini 截图问 → 🎨 海报/视觉 → ChatGPT 生图 → 📋 批量任务 → Agent 自动跑 → 🏗️ 整合交付 → Claude Code
快速上手

如果你是第一次接触 AI 工具，从这里开始

1
注册 ChatGPT — 先让海报设计变得轻松。简单 prompt 就能出不错的图，慢慢学会跟它打磨。
2
装上 Gemini — 免费、快、不限量。工作中遇到任何不确定的事，截图问她。
3
试着串联 — Gemini 给你方案 → ChatGPT 出视觉物料 → 你拿着结果交付。这就已经是 AI 工作流了。
4
想更进一步？ — 找团队里用过 Claude Code 的同事帮忙跑一遍安装脚本，看看复杂任务自动化是什么样的。
